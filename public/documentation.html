My Movies

This web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, create a list of their favorite movies as well as a to watch list .

Action Request URL Response
Return all movies GET '/movies' Returns an array of all JSON movie objects in the database
Return single movie GET '/movies/:title' Returns a JSON object holding data for a single movie
Return data about a genre GET '/movies/:genre' Returns an array of all JSON movie objects with the genre by movie name as well as genre description
Return data about a director GET '/movies/:directorName' Returns an array of all JSON movie objects with the director by movie name as well as bio of director
New User POST /newUser Allow new users to register.
Allow users to update their user info PUT '/users/:username' Returns the data fields that was updated by the user
Allow users to add a movie to their list of favorites PUT '/users/add/:movieName' Returns a text confirming the movie was added successfully
Allow users to remove a movie to their list of favorites DELETE '/users/remove/:movieName' Returns a text confirming which movie was removed successfully
Allow existing users to deregister DELETE '/users/deleteAccount' Returns a text confirming that the user was reregistered
